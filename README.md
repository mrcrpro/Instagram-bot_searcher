# Instagram-bot_searcher
Program to search for information about an Instagram user, with or without logging into an account.

The information displayed for each user is:
- Username
- ID
- Full name
- Followers
- Following
- Posts
- External URL
- Verified account
- Business account
- Private account
- Bio

# Additional Features
- Available in three languages: Spanish, English, and German.

# Issues detected
- An error occurs when attempting to obtain information about an account's followers and following.
